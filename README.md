# Cinema Booking Service (Coursework)

Це навчальний проєкт — сервіс бронювання квитків для кінотеатрів. Містить REST API для роботи з фільмами, сеансами, тарифами та бронюваннями. Проєкт реалізований на Node.js з використанням TypeORM і PostgreSQL.

Технологічний стек
------------------
- **Мова:** Node.js (рекомендовано v18+)
- **ORM:** TypeORM
- **БД:** PostgreSQL
- **Фреймворк HTTP:** Express
- **Тестування:** Jest + Supertest

Інструкції з налаштування
-------------------------
1. Клонуйте репозиторій:

	 `git clone https://github.com/Fererra/coursework.git`

2. Перейдіть в каталог проєкту:

	 `cd coursework`

3. Встановіть залежності (за бажанням локально):

	 `npm install`

4. Налаштуйте змінні середовища у файлі `.env` (створіть на основі прикладу)

5. Запустіть сервіси:

	 `npm start`  # скрипт запускає docker compose

Запуск тестів
-------------
- Запустити всі тести:

	`npm test`

- Запустити конкретний файл тестів (наприклад):

	`npm test -- tests/integration/auth.spec.js`

Огляд структури проєкту
------------------------
- `src/` — основний код програми
	- `server.js`, `app.js` — точка входу
	- `common/` — утиліти та константи
	- `database/` — налаштування TypeORM, сутності, міграції, репозиторії
	- `modules/` — бізнес-логіка по модулях (auth, movie, showtime, booking тощо)
- `tests/` — інтеграційні тести

Приклади API / використання
---------------------------
- Отримати список фільмів (припустимий ендпоінт):

	`GET http://localhost:3000/api/movies`

- Увійти (припустимий ендпоінт):

	`POST http://localhost:3000/api/auth/login`

	Тіло (JSON):

	```json
	{
		"email": "user@example.com",
		"password": "password"
	}
	```
